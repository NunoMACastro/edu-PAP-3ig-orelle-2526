# osmeticAI

Aplicação de cosmética inteligente que analisa fotografias do rosto, identifica necessidades da pele e sugere produtos personalizados com base em IA.

## Objetivo

Criar um sistema de consultoria cosmética digital, com análise automática da pele e recomendações otimizadas.

---

## Documentação Técnica

-   [Requisitos Funcionais](docs/RF.md)
-   [Requisitos Não Funcionais](docs/RNF.md)

---

## Funcionalidades Principais

-   Upload de fotografias do rosto.
-   IA para detetar acne, oleosidade, manchas e rugas.
-   Relatórios personalizados.
-   Recomendação inteligente de produtos.
-   Simulação virtual (maquilhagem/antes-depois).
-   Catálogo de produtos detalhado.
-   Carrinho de compras, pagamentos e histórico.
-   Painel administrativo.
-   Chat cliente ↔ consultor (opcional).

---

## Roadmap

-   MVP: análise de fotografia + catálogo.
-   Simulação virtual + recomendações avançadas.
-   Chat com consultor.
